Function Name, Syntax Usage Example Usage in Project
  • np.array(object)
  • Creates a NumPy array (ndarray) from a list, tuple, or other array-like structure.
  • Supports vectorized operations (no need to loop over the elements of an ndarray to perform calculations on them.).
  • Includes methods for matrix calculations.
  • Used instead of Python lists for efficiency.
  • np.array([[1, 2], [3, 4]])
  • Creates a 2×2 NumPy array.
  • Creates 2D input/output matrices for training.
  • np.exp(array)
  • Computes e^(array) element-wise.
  • np.exp(np.array([0, 1]))
    Output: [1. 2.71828183]
  • Used in sigmoid activation.
  • np.random.rand(rows, cols)
  • Creates a matrix of shape (rows, columns) with random values ∈ [0, 1).
  • np.random.rand(2, 3)
  • Generates 2×3 matrix of random floats.
  • Used to initialize weights randomly.
  • np.zeros((rows, cols))
  • Creates a matrix filled with zeros.
  • np.zeros((2, 3))
    Output: [[0. 0. 0.]
    [0. 0. 0.]]
  • Initializes biases to 0.
  • np.dot(A, B)
  • Performs dot product between 2 arrays.
  • A = np.array([[1, 2]])
    B = np.array([[3], [4]])
    np.dot(A, B)
    Output: [[11]]
  • Used to compute forward pass values.
  • np.sum(array, axis, keepdims=True)
  • Sums along the specified axis (axis=0 for vertical axis, axis=1 for horizontal axis).
  • keepdims=True preserves the original number of dimensions in the result.
  • a = np.array([[1, 2], [3, 4]])
    np.sum(a, axis=0, keepdims=True)
    Output: [[4 6]]
  • Used to update biases during backpropagation.
  • np.mean(x)
  • Computes the average of elements.
  • np.mean([2, 4, 6])
    Output: 4.0
  • Used to compute Mean Squared Error (MSE) loss.
  • np.square(x)
  • Returns element-wise squares.
  • np.square(np.array([2, -3]))
    Output: [4 9]
  • Used to compute squared error in loss.
  • np.round(x, decimals=n)
  • Rounds each value in the array to n decimal places.
  • np.round([1.234, 5.678], 2)
    Output: [1.23, 5.68]
  • Used when printing predictions.
