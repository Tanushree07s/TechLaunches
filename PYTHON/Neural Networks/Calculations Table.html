Usage Formula
  • Output Layer Gradient: Measures how much the final error changes with respect to the output neuron's input.
  • final error × sigmoid derivative at output
  • Hidden Layer Gradient: Determines each hidden neuron's contribution to the final error by:
    • Propagating the output error backward through the output weights.
    • Scaling the result using the sigmoid derivative of the hidden neuron's output.
  • Transpose the weights from hidden to output layer from shape (2, 1) to (1, 2).
  • Use weights_hidden_output.T to transpose the matrix.
  • output layer gradient · transposed weights × sigmoid derivative of hidden output
  • Updating Weights (Hidden → Output)
  • old weight + (hidden output.T · output error) × learning rate
  • Updating Output Biases
  • old bias + total output error × learning rate
  • Total output error is calculated using np.sum(d_output, axis=0, keepdims=True)
  • Updating Weights (Input → Hidden)
  • old weight + (input.T · hidden error) × learning rate
  • Updating Hidden Biases
  • old bias + total hidden error × learning rate
  • Total hidden error is calculated using np.sum(d_hidden, axis=0, keepdims=True)
  • Loss Function: Aggregates all final errors in the network.
  • Mean Squared Error (MSE) of final errors during training
