Function Name, Syntax Usage Usage in Project
  • TfidfVectorizer()
  • Initializes a TF-IDF vectorizer to convert text data into numerical feature vectors.
  • Used to prepare question data for vector comparison in the retrieval-based chatbot.
  • vectorizer.fit_transform(list)
  • Fits the TF-IDF vectorizer on a list of text entries and transforms them into a TF-IDF matrix.
  • Applies TF-IDF to the list of predefined questions so they can be compared with user input.
  • cosine_similarity(vector_a, vector_b)
  • Calculates cosine similarity scores between two vectors or matrices.
  • Used to compare the vectorized user input with all stored question vectors to find the closest match.
  • similarity_scores.argmax()
  • Returns the index of the maximum value in a similarity score array.
  • Identifies the index of the most similar known question to generate the best response.
