1. GPT Models (GPT-1 to GPT-4o):
    • Trained on a huge amount of internet data to learn grammar, facts, and reasoning.
    • GPT-4 understands both text and images, and GPT-4o adds voice and sound too.
  2. Making AI Behave Better (RLHF):
    • Used feedback from real people to teach the model how to give safer, more helpful answers.
    • Combined reward systems and tuning methods to align AI behavior with human values.
  3. ChatGPT and API Access:
    • Launched ChatGPT and APIs that let anyone use these models in apps and websites.
    • Handles tasks like writing summaries, answering questions, translating text, or writing code.

  1. BERT:
    • Uses bidirectional attention to read both left and right context simultaneously, enabling deep understanding of word relationships in a sentence.
    • Learns to predict randomly masked words using surrounding context (Masked Language Modeling) , producing contextual word embeddings (word meanings that depend on neighboring words).
    • Pretrained to guess missing words and understand how sentences relate to each other (Next Sentence Prediction) .
  2. T5 (Text-to-Text Transfer):
    • It is a large transformer model that treats every NLP task as a text-to-text problem. This means both the input and output are plain text, regardless of the task. For instance, for an English-to-Spanish translation, the input could be: "translate English to German: That is a table." then the output would be the translated sentence in Spanish.".
    • Trained on a huge cleaned dataset called C4 using a method called span corruption .
  3. Built PaLM and Gemini models for even smarter, multi-modal AI (text, image, video).
  4. Flan-T5 is a version that was fine-tuned to handle a wide range of NLP tasks such as translation, question answering,reasoning, and text generation by framing them all into a text-to-text format.

  1. XLM & XLM-R:
    • Multilingual models trained on 100+ languages using a technique called masked language modeling.
    • Can do cross-lingual tasks (e.g., translate or understand languages it wasn’t directly trained on).
  2. fairseq & LASER:
    • fairseq: A tool to train translation and summarization models.
    • LASER: Converts sentences from any language into a numerical vector embeddings in a shared multilingual space Instead of each language having its own separate space of vectors, LASER makes one single space where sentences from different languages that mean the same thing end up close to each other. . This common form allows direct comparison across languages—for example, a sentence in English can be matched with its equivalent in French or Spanish by comparing their embeddings.
  3. M2M-100:
    • M2M-100: A translation model that translates directly between 100 languages (no English in-between).

  1. Copilot (powered by OpenAI's Codex model ):
    • Built into VS Code and GitHub to suggest code while you type.
    • Trained on billions of lines of code to understand developer intent.
  2. Azure OpenAI Service:
    • Offers GPT models for businesses through Azure with privacy and security features.
    • Used to power Microsoft Word, Excel, and Teams via "Copilot" features.
  3. DeepSpeed & ONNX Runtime:
    • DeepSpeed: Helps train massive models faster and with less memory.
    • ONNX Runtime: Makes models run efficiently across different devices.
  4. Responsible AI Tools:
    • Microsoft builds tools like Fairlearn and InterpretML to test for bias and explain model decisions.
    • Focused on building safe, ethical NLP systems for real-world use.
